GtkTreeIter

Description [src] 

struct GtkTreeIter {
  int stamp;
  gpointer user_data;
  gpointer user_data2;
  gpointer user_data3;
}

The GtkTreeIter is the primary structure for accessing a GtkTreeModel. Models are expected to put a unique integer in the stamp member, and put model-specific data in the three user_data members.
