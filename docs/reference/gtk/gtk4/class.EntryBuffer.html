Class

GtkEntryBuffer

Description [src] 

class Gtk.EntryBuffer : GObject.Object {
  parent_instance: GObject
}

A GtkEntryBuffer hold the text displayed in a GtkText widget.

A single GtkEntryBuffer object can be shared by multiple widgets which will then share the same text content, but not the cursor position, visibility attributes, icon etc.

GtkEntryBuffer may be derived from. Such a derived class might allow text to be stored in an alternate location, such as non-pageable memory, useful in the case of important passwords. Or a derived class could integrate with an applicationâ€™s concept of undo/redo.
