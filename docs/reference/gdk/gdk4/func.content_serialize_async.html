Description [src]

Serialize content and write it to the given output stream, asynchronously.

The default I/O priority is G_PRIORITY_DEFAULT (i.e. 0), and lower numbers indicate a higher priority.

When the operation is finished, callback will be called. You must then call gdk_content_serialize_finish() to get the result of the operation.

Parameters

 The data is owned by the caller of the function.

 The string is a NUL terminated UTF-8 string.
